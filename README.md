# JobTracker

**JobTracker** is a desktop application built in C# designed to help you track job applications, recruiters, interview status, and notes in one convenient place.

---

## Features

- Track job applications by company, position, and status.  
- Add notes for each application.  
- Keep a list of contacts (recruiters, hiring managers) for each application.  
- Automatic SQLite database management with EF Core migrations.  
- Simple, easy-to-use GUI interface.
